Terminal utility to manage Transmission BitTorrent Client.

> Currently not on a working stage

## Environment

Clone the repo
```
$ git clone git@github.com:tvillega/gearbox-lever.git
```

Create python environment:
```
$ cd gearbox-lever
$ python -m venv .env
$ . .env/bin/activate
(.env) $ pip install transmission-rpc
```
